# BroxLab AI - ব্যবহার নির্দেশিকা 🚀

BroxLab AI হলো একটি ওপেন-সোর্স অটোনোমাস কোডিং অ্যাসিস্ট্যান্ট যা সরাসরি আপনার VS Code এর অ্যাক্টিভিটি বার থেকে কাজ করে! এটি OpenRouter API ব্যবহার করে আপনার কোড বুঝতে, লিখতে, টার্মিনাল কমান্ড রান করতে এবং এমনকি রিয়েল-টাইমে ওয়েব সার্চ করতে পারে। এর ডিজাইন জনপ্রিয় এক্সটেনশনগুলোর (যেমন: GitHub Copilot বা Cline) আদলে তৈরি।

---

## 🌟 ফিচারসমূহ (Features)

- **সম্পূর্ণ স্বাধীন (Autonomous):** ব্রক্সল্যাব এআই নিজে থেকেই আপনার প্রজেক্টের ফাইল রিড, রাইট ও ডিলিট করতে পারে।
- **টার্মিনাল কমান্ড:** এটি সরাসরি টার্মিনালে কমান্ড রান করে বিল্ড বা টেস্ট করতে পারে।
- **ওয়েব সার্চ প্লাগইন:** লেটেস্ট ডকুমেন্টেশন বা সলিউশন খুঁজতে এটি রিয়েল-টাইম ইন্টারনেট সার্চ করতে পারে!
- **অ্যাডভান্সড সেটিংস প্যানেল:** UI-তে থাকা সেটিংস ট্যাব ব্যবহার করে খুব সহজেই API Key ও Model পরিবর্তন করা যায়।
- **ক্লিয়ার চ্যাট:** এক ক্লিকেই চ্যাট হিস্ট্রি মুছে নতুন কাজ শুরু করতে পারবেন।
- **প্রোঅ্যাক্টিভ বাগ ডিটেকশন:** সেটিংসে "Proactive Bug Detection" চালু করলে এআই স্বয়ংক্রিয়ভাবে কোডে বাগ খুঁজে বের করবে।
- **কোডলেন্স (CodeLens):** ফাংশন ও ক্লাসের উপরে "Explain", "Refactor", "Fix" অ্যাকশন বাটন দেখা যাবে।

## 🛠️ সেটআপ এবং ইন্সটলেশন (Installation)

1. আপনার VS Code-এ ফোল্ডারটি (`h:\vs-code-ex\broxlab-ai`) খুলুন।
2. যদি ডিপেন্ডেন্সি ইন্সটল করা না থাকে, তাহলে টার্মিনাল খুলে টাইপ করুন:
   ```bash
   npm install
   ```
3. এবার এক্সটেনশনটি কম্পাইল করতে হবে:
   ```bash
   npm run compile
   ```
4. এরপর কিবোর্ড থেকে **`F5`** চাপুন। এটি একটি নতুন "Extension Development Host" উইন্ডো খুলবে।
5. বাম দিকের অ্যাক্টিভিটি বারে **"Sparkle" (তারকা)** আইকনটি দেখতে পাবেন। সেখানে ক্লিক করলেই BroxLab AI এর ইউজার ইন্টারফেস ওপেন হবে!

## 🚀 রিলিজ ও সংস্করণ (Release Workflow)

### দ্রুত রিলিজ করার স্টেপ:
1. বিল্ড করুন (ওয়েবভিউ + কোড):
   ```bash
   npm run build-webview
   ```
2. লিন্ট চালান (টাইপস্ক্রিপ্ট ক্লিন):
   ```bash
   npm run lint
   ```
3. রিলিজ করুন (বাম্প + ট্যাগ + পুশ):
   ```bash
   npm run release [patch|minor|major|prerelease|none]
   ```
   - যদি আপনি কোনও অর্গুমেন্ট না দেন, `patch` ডিফল্ট হবে।
   - `none` দিলে সংস্করণ বাম্প হবে না; শুধু কমিট ও ট্যাগ পুশ হবে।

## ⚙️ কনফিগারেশন (Configuration)

ব্যাবহার শুরু করার জন্য আপনাকে OpenRouter-এর API কি ও ব্যবহারের মডেল নির্ধারণ করে দিতে হবে।

1. **Settings Tab**-এ যান।
2. **OpenRouter API Key** বক্সে আপনার OpenRouter অ্যাকাউন্ট থেকে পাওয়া API কি দিন।
3. **Model** বক্সে আপনি কোন মডেলটি ব্যবহার করতে চান তা লিখুন (যেমন: `anthropic/claude-3.7-sonnet` অথবা `openai/gpt-4o`)।
4. **Save Settings** বাটনে ক্লিক করুন। আপনার কি (Key) এখন নিরাপদে VS Code Secrets-এ জমা হয়ে গেছে।

### অতিরিক্ত সেটিংস:

- **Local Models (Ollama):** ব্রক্সল্যাব লোকাল মডেলও সাপোর্ট করে।
  1. [Ollama](https://ollama.com) ইন্সটল করুন
  2. টার্মিনালে `ollama serve` রান করুন
  3. এক্সটেনশনটি অটো-ডিটেক্ট করবে
  4. সেটিংসে `ollama/` দিয়ে শুরু মডেল সিলেক্ট করুন

- **Approval Mode:** 
  - `Ask Every Time` (নিরাপদতম) - প্রতিটি অ্যাকশনের আগে অনুমতি চায়
  - `Ask Once Per Session` - একবার অনুমতি দিলে সেশনে সব অ্যাকশন হয়
  - `Auto Approve` (দ্রুততম) - সব অ্যাকশন স্বয়ংক্রিয়ভাবে হয়

- **Proactive Bug Detection:** `false` থেকে `true` করলে প্রতিটি সেভের পর এআই অটোম্যাটিক্যালি বাগ খুঁজবে।

- **Enable CodeLens:** `true` করলে ফাংশনের উপর এক্সপ্লেইন, রিফ্যাক্টর, ফিক্স বাটন দেখাবে।

## 💡 কিভাবে ব্যবহার করবেন (How to Use)

সেটিংস কনফিগার করার পর, **"Chat"** ট্যাবে ফিরে আসুন। এখন এটি ব্যবহার করার জন্য প্রস্তুত!

### কিছু চমৎকার প্রম্পটের উদাহরণ:

1. **কোড লেখা:** 
   > "এই ওয়ার্কস্পেসে একটি index.html ফাইল এবং script.js ফাইল তৈরি করো যেখানে একটি সাধারণ টু-ডু লিস্ট অ্যাপ থাকবে।"
2. **টার্মিনাল কমান্ড রান করা:** 
   > "npm install express কমান্ডটি রান করো।"
3. **ফাইল মুছে ফেলা:** 
   > "আমার প্রজেক্ট থেকে index.html ফাইলটি ডিলিট করে দাও।"
4. **ইন্টারনেট সার্চ করা:** 
   > "OpenRouter এর নতুন Web Search API কিভাবে কাজ করে তা ইন্টারনেটে খুঁজে দেখো এবং আমাকে জানাও।"
5. **কোড রিফ্যাক্টর ও পড়া:** 
   > "src ফোল্ডারে থাকা agent.ts ফাইলটি পড়ো এবং সেখানে delete_file টুলটি কিভাবে কাজ করছে তা আমাকে বুঝিয়ে দাও।"

**চ্যাট ক্লিয়ার করা:** যদি আপনি নতুন কোনো প্রজেক্ট বা প্রসঙ্গে কথা বলতে চান, তবে ডানদিকে থাকা **"Clear Chat"** বাটনে প্রেস করলেই আগের সব কথাবার্তা মুছে যাবে।
